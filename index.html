Opt Outdoors ATX
Opt Outdoors ATX

Opt Outdoors ATX is a single page web application for outdoor enthusiasts to organize and join outdoor activities in the Austin area. This web app is my Galvanize capstone project and was also my first time using React.


React and Bootstrap were used on the frontend while Node, Express and PostgreSQL make up the back end. The site is currently deployed on Heroku.

Console.Music
Console.Music

Console.Music is a web application for users to share and explore Spotify tracks posted by others. This web app was built during a one-week sprint and was the first full-stack application I built at Galvanize.


The app utilizes the Spotify API and the passport-spotify library for authentication. EJS and Materialize were used for templating and styling, respectively, while Node, Express and PostgreSQL make up the back end.
